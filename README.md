# RBX-Inventory-System

## Status
**Experimental**  
**Versão:** v0.1.0  

---

## Visão Geral

Sistema básico de inventário baseado em slots, desenvolvido em Lua para a plataforma Roblox.

Esta versão representa a estrutura inicial do sistema, com foco na criação dinâmica de slots e organização modular através do `ConfigModule` (ModuleScript).

O sistema ainda não é utilizável nesta versão, pois se trata de um protótipo estrutural experimental.

---

## Funcionalidades Atuais (v0.1.0)

### Função Principal: `CreateSlots()`

- Criação dinâmica de slots com base na configuração `MaxSlots` definida em `Slots_Config`
- Modo de depuração configurável (`debugMode` — boolean)
- Controle simples de exibição dos slots
- Estrutura modular inicial
